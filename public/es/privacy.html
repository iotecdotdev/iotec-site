Política de Privacidad

Última actualización: 2026-05-17

1. Quiénes somos

iotec es un desarrollador independiente de software construyendo herramientas para NinjaTrader 8. Contacto: soporte@iotec.dev.

2. Datos que recolectamos

2.1 Del indicador (durante uso normal)

  • License key — para validar tu suscripción.
  • Device fingerprint — un hash SHA-256 anónimo derivado de tu MachineGuid + serial BIOS + serial del motherboard. Es opaco y no se puede revertir para identificar detalles de tu hardware.
  • Assembly hash — SHA-256 de los DLLs iotec instalados, para detección de tampering.
  • IP pública — visible para nosotros, como en cualquier servicio web.
  • Timestamp de cada activación y heartbeat.

2.2 Del sitio web / compra

  • Correo y nombre — proporcionados en checkout via Lemon Squeezy.
  • Datos estándar del navegador — user agent, referrer, etc. cuando visitas iotec.dev (sin trackers terceros, sin scripts de analytics en este momento).

3. Datos que NO recolectamos

El indicador nunca envía a nuestros servidores:

  • Tus posiciones, órdenes abiertas, P&L o balance de cuenta.
  • Los instrumentos que operas o vigilas.
  • Configuraciones de chart o data de workspace.
  • Tus credenciales de broker.
  • Cualquier contenido de tus trades o estrategias.

4. Dónde vive la data

  • Servidor de licencias: hospedado en Europa (Contabo, Alemania).
  • Servicio de email: Brevo (Sendinblue, basado en EU).
  • Procesador de pagos: Lemon Squeezy (US, Merchant of Record compatible con GDPR).
  • CDN/DNS: Cloudflare.
  • Backups: Cloudflare R2 (cifrados en reposo).

5. Data local en tu computadora

El indicador guarda tu license key localmente en %APPDATA%\iotec\license*.dat, cifrada con Windows DPAPI atada a tu cuenta de Windows. Solo tú, con tu sesión iniciada, puedes descifrarla.

6. Cómo usamos la data

  • Para validar tu licencia y prevenir uso no autorizado.
  • Para enviarte license keys, respuestas de soporte y actualizaciones de producto.
  • Para detectar tampering o abuso.
  • Para mejorar el producto (métricas agregadas, anónimas).

No vendemos, alquilamos ni compartimos tu data con terceros para marketing.

7. Tus derechos (GDPR / leyes similares)

Tienes derecho a:

  • Solicitar una copia de tu data.
  • Solicitar corrección o eliminación.
  • Retirar consentimiento.
  • Presentar queja ante tu autoridad local de protección de datos.

Envía correo a soporte@iotec.dev para cualquier solicitud.

8. Retención

  • Licencia + correo: mientras tu licencia esté activa + 12 meses tras la última actividad.
  • Logs del servidor: 90 días.
  • Backups: 90 días.

9. Seguridad

Usamos TLS para toda comunicación, backups cifrados, payloads de licencia firmados con RSA, y servidores con acceso restringido. Ningún sistema es 100% seguro; hacemos lo mejor posible.

10. Cambios

Podemos actualizar esta política. Cambios materiales se anunciarán por correo a licenciatarios activos cuando sea posible.

11. Contacto

Dudas o solicitudes de data: soporte@iotec.dev
